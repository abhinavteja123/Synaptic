### Prerequisites

| Requirement | Where to Get It |
|-------------|----------------|
| **Node.js 18+** | [nodejs.org](https://nodejs.org) |
| **Groq API Key** | [console.groq.com](https://console.groq.com) (free) |

### 1. Clone & Install

```bash
git clone https://github.com/your-username/synaptic.git
cd synaptic
npm install
```

### 2. Set Up Supabase

1. Create a free project at [supabase.com](https://supabase.com)
2. Go to **SQL Editor** â†’ paste and run the entire contents of `supabase-schema.sql`
3. Go to **Settings â†’ API** â†’ copy your **Project URL** and **anon/public key**

### 3. Configure Environment

```bash
cp .env.example .env.local
```

Edit `.env.local` with your values:

```env
# Groq AI (free at https://console.groq.com)
GROQ_API_KEY=gsk_your_groq_api_key_here

# JWT Secret (generate with: openssl rand -base64 32)
JWT_SECRET=your_random_secret_string

# Supabase
NEXT_PUBLIC_SUPABASE_URL=https://your-project.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key

# PartyKit (optional, for multiplayer)
NEXT_PUBLIC_PARTYKIT_HOST=localhost:1999

# App URL
NEXT_PUBLIC_APP_URL=http://localhost:3000
```

### 4. Run Development Server

```bash
# Terminal 1: Start Next.js
npm run dev
```

```bash
# Terminal 2 (optional): Start PartyKit multiplayer server
npx partykit dev
```

Open **[http://localhost:3000](http://localhost:3000)** and start creating memories!

### 5. Production Build

```bash
npm run build
npm start
```
